# Task List App

Минимальное Flutter-приложение для создания, отображения, редактирования и удаления задач (CRUD).

Файлы:
- `lib/main.dart` - главный код приложения.
- `pubspec.yaml` - зависимости.

Как запустить (требуется установленный Flutter SDK):

1. Откройте терминал в папке проекта (где находится `pubspec.yaml`).

2. Получите зависимости:

```bash
flutter pub get
```

3. Запустите на подключённом устройстве или эмуляторе:

```bash
flutter run
```
```

Приложение использует `StatefulWidget`, `TextField` и `ListView.builder`.

Функциональность:
- Введите задачу в поле и нажмите Add или Enter, чтобы добавить.
- Нажмите иконку редактирования, чтобы открыть диалог и изменить текст задачи.
- Нажмите иконку удаления, чтобы удалить задачу (с подтверждением).

Замечания:
- Данные не сохраняются между запусками (для этого можно добавить локальное хранилище, например `shared_preferences`).
